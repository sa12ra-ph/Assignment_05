Q1:What is the difference between var, let, and const?
Ans:
*Var is Block-scoped. It can be re-declared but not re-assigned.
*let is block-scoped. It can be re-assigned but not re-declared in the same scope.
*const is block-scoped. It can't be re-assigned or re-declared

Q2:What is the spread operator (...)?
Ans:
*Spread operator is used to expand elements of an array or object.It is commonly used for copying arrays, merging arrays and copying objects.

Q3:What is the difference between map(), filter(), and forEach()?
Ans:
*map creates a new array by transforming each element.
*filter creates a new array with elements that match a condition.
*forEach loops through an array but doesn't return a new array.

Q4:What is an arrow function?

Q5:What are template literals?
Ans:
*Template literals is strings using backticks.It allows embedding variables using ${} and supports multi-line strings.
